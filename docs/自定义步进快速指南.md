# 自定义步进快速指南

## 🎯 一句话说明

**在"步进"框输入数字，按钮就会每次跳那么多帧！**

## 📺 界面说明

```
00:01:23 步进:[5] [◀-5帧] ━━━◉━━━ [+5帧▶] 00:05:30
         ↑      ↑                    ↑
       这里输入  向前5帧            向后5帧
```

## 🚀 三秒上手

### 1️⃣ 输入步进值
```
在"步进"框输入数字（如：10）
```

### 2️⃣ 按钮自动变化
```
按钮变为 [◀-10帧] 和 [+10帧▶]
```

### 3️⃣ 点击跳转
```
每次点击跳10帧，速度提升10倍！
```

## 💡 常用设置

| 步进值 | 效果 | 适用场景 |
|--------|------|---------|
| 1 | 逐帧 | 精确定位 ✨ |
| 5 | 快5倍 | 微调 |
| 10 | 快10倍 | 快速浏览 |
| 25 | 约1秒 | 按秒跳转 (25fps) |
| 50 | 约2秒 | 超快浏览 |

## 🎬 实用示例

### 示例1: 快速跳过片头
```
1. 步进设为 30
2. 连续点 [+30帧▶]
3. 快速跳过30秒片头
```

### 示例2: 精确定位
```
1. 步进设为 10，快速接近
2. 改为步进 1，精确定位
3. 找到完美的一帧
```

### 示例3: 浏览长视频
```
1. 步进设为 50
2. 连续点击快速浏览
3. 发现目标立即停止
```

## 📊 推荐值

### 25fps 视频
- **步进 1** = 40ms/帧（精确）
- **步进 25** = 1秒
- **步进 50** = 2秒

### 30fps 视频
- **步进 1** = 33ms/帧（精确）
- **步进 30** = 1秒
- **步进 60** = 2秒
