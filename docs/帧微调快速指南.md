# 帧微调按钮快速指南

## 🎯 三步上手

### 1️⃣ 选择视频
```
点击"选择视频文件" 
→ 帧微调按钮自动启用
```

### 2️⃣ 粗略定位
```
拖动时间轴滑块
→ 接近目标位置
```

### 3️⃣ 精确微调
```
点击 [◀-1帧] 向前一帧
点击 [+1帧▶] 向后一帧
→ 逐帧精确查找
```

## 🖼️ 界面说明

```
00:01:23  [◀-1帧] ━━━━━━◉━━━━━━ [+1帧▶]  00:05:30
          ↑                      ↑
       向前一帧               向后一帧
```

## 💡 实用技巧

### ✅ 组合使用
```
滑块粗定位 + 按钮精调 = 完美定位
```

### ✅ 逐帧浏览
```
连续点击 [+1帧▶] = 慢动作播放效果
```

### ✅ 找清晰帧
```
跳过模糊帧 → 找到静止帧 → 最佳预览效果
```

## 📊 精度对比

| 方法 | 精度 | 适用场景 |
|------|------|---------|
| 滑块 | 秒级 | 快速定位 |
| 按钮 | 帧级 | 精确微调 |

## ⚙️ 常见帧率

| 帧率 | 一帧时长 | 类型 |
|------|---------|------|
| 24 fps | ~42ms | 电影 |
| 25 fps | 40ms | PAL |
| 30 fps | ~33ms | 常见 |
| 60 fps | ~17ms | 高帧率 |

## 🎬 使用示例

### 示例1: 找人物最佳表情
```
1. 滑块定位到人物出现
2. 点击 [+1帧▶] 逐帧查看
3. 找到最佳表情帧
4. 调整字幕
```

### 示例2: 避开台标/水印
```
1. 滑块定位到有台标区域
2. 微调 [◀-1帧] / [+1帧▶]
3. 找到台标位置变化
4. 调整字幕位置
```

### 示例3: 跳过转场特效
```
1. 滑块定位到某位置
2. 发现是黑屏/转场
3. 点击 [+1帧▶] 跳过
4. 找到正常画面
```

## ⚠️ 注意事项

1. **响应时间**
   - 每次点击需要1-2秒提取帧
   - 不要连续快速点击

2. **帧率识别**
   - 自动识别视频真实帧率
   - 特殊格式可能使用默认25fps

3. **边界限制**
   - 自动限制在视频范围内
   - 不会超出视频开头/结尾

## 🚀 完整工作流

```
1. 选择视频
   ↓
2. 滑块拖到大概位置
   ↓
3. [+1帧▶] 向后微调
   或
   [◀-1帧] 向前微调
   ↓
4. 找到最佳帧
   ↓
5. 调整字幕参数
   ↓
6. 实时预览效果
```

## 🎯 功能亮点

- ⭐ **帧级精确**: 精确到单个视频帧
- ⭐ **自动帧率**: 自动识别视频真实帧率
- ⭐ **逐帧浏览**: 连续点击逐帧查看
- ⭐ **滑块同步**: 按钮和滑块自动同步
