# LLM 字幕翻译 - 快速开始

## 5分钟上手指南

### 第一步：准备 API Key

选择一个 LLM 服务商并获取 API Key：

- **OpenAI** (推荐新手): https://platform.openai.com/api-keys
- **DeepSeek** (国内可用): https://platform.deepseek.com/
- **Claude**: https://console.anthropic.com/
- **Gemini**: https://makersuite.google.com/app/apikey

### 第二步：启动程序

```bash
python main.py
```

### 第三步：打开翻译工具

在主菜单点击 **"🌍 AI字幕翻译"**

### 第四步：配置参数

1. **LLM提供商**: 选择 OpenAI（或其他）
2. **API Key**: 输入你的密钥
3. **模型**: 选择 gpt-4o-mini（经济实惠）
4. **目标语言**: 选择要翻译成的语言

### 第五步：选择文件

点击 **"📁 选择字幕文件"**，选择你的 .srt 文件

### 第六步：开始翻译

点击 **"🚀 开始翻译"**，等待完成！

## 推荐配置

### 经济实惠型（适合新手）
```
提供商: OpenAI
模型: gpt-4o-mini
批次大小: 10
```
**特点**: 便宜快速，质量不错

### 高质量型（适合专业用户）
```
提供商: OpenAI
模型: gpt-4o
批次大小: 5
```
**特点**: 翻译质量最高，费用较高

### 国内友好型
```
提供商: DeepSeek
模型: deepseek-chat
批次大小: 10
```
**特点**: 国内可直接访问，性价比高

## 提示与技巧

### ✅ 推荐做法
- 首次使用时选择较小的字幕文件测试
- 使用 gpt-4o-mini 进行初步翻译
- 重要内容使用 gpt-4o 翻译
- 设置 API 费用上限

### ❌ 避免做法
- 不要在公共网络输入 API Key
- 不要使用过大的批次大小（>20）
- 不要忽略错误提示
- 不要在未测试前处理大量文件

## 结果位置

翻译完成的文件保存在：
```
~/Videos/pyvideotrans/LLMTranslate/
```

文件命名格式：
```
原文件名_translated_目标语言.srt
```

例如：
```
movie_translated_zh.srt  (翻译成中文)
movie_translated_en.srt  (翻译成英文)
```

## 遇到问题？

1. **无法连接 API**
   - 检查网络连接
   - 尝试配置代理
   - 确认 API Key 正确

2. **翻译质量不好**
   - 减小批次大小
   - 更换更好的模型
   - 指定源语言

3. **费用问题**
   - 使用 gpt-4o-mini 而不是 gpt-4o
   - 在服务商后台设置费用上限
   - 先用小文件测试
