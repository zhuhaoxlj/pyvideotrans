# 时间轴滑块使用指南

## 🎯 功能简介

时间轴滑块让您可以**自由选择**视频的任意时间点进行字幕预览，而不是只能看中间那一帧！

## 📺 界面说明

```
┌─────────────────────────────────────┐
│      视频预览区域                    │
│      (显示当前选中时间点的画面)       │
└─────────────────────────────────────┘
  00:01:23  ━━━━━━━━◉━━━━━━━━  00:05:30
  ↑         ↑                    ↑
  当前时间  拖动滑块            总时长
           [  刷新预览  ]
```

## 🚀 三步上手

### 1️⃣ 选择视频
点击"选择视频文件"，选好后会看到：
- ✅ 右侧显示视频总时长（如 00:05:30）
- ✅ 滑块自动启用
- ✅ 预览显示视频中间的画面
- ✅ 左侧显示当前时间

### 2️⃣ 拖动滑块
用鼠标拖动滑块到您想看的位置：
- 往左拖 = 看视频前面的画面
- 往右拖 = 看视频后面的画面
- 点击轨道 = 直接跳到那个位置

### 3️⃣ 调整字幕
找到合适的画面后，调整字幕参数：
- 位置、大小、颜色等
- 预览会自动更新
- 实时看到字幕效果

## 💡 实用技巧

### 技巧1：找到最好的预览位置
不同场景的光线、背景不同，字幕效果也不同：

```
暗色场景（0:30）→ 测试亮色字幕是否清晰
亮色场景（2:15）→ 测试深色轮廓是否明显
人物特写（3:45）→ 测试字幕不会挡住重要内容
```

### 技巧2：快速定位
- **开头**: 拖到最左边
- **中间**: 默认就是中间，或拖到50%附近
- **结尾**: 拖到最右边
- **具体位置**: 看左侧时间标签，拖到目标时间

### 技巧3：对比不同时间点
```
1. 拖到00:30，调整字幕（暗场景）
2. 拖到02:45，检查效果（亮场景）  
3. 拖到04:10，最终确认（中间调）
4. 确保字幕在所有场景都清晰
```

## ⚠️ 注意事项

### ✅ 可以做的
- ✅ 随意拖动滑块
- ✅ 多次拖动选择不同位置
- ✅ 边拖动边调整字幕
- ✅ 拖到0%看开头，100%看结尾

### ❌ 注意事项
- ⚠️ 快速拖动时要等一下，让视频帧加载
- ⚠️ 每次拖动都会重新截取视频帧
- ⚠️ 预览使用的是示例文字，不是您的真实字幕内容

## 🐛 常见问题

### Q: 滑块是灰色的，拖不动？
**A:** 您还没选择视频文件，先点"选择视频文件"

### Q: 拖动后画面没变化？
**A:** 稍等几秒，系统在提取视频帧。大视频可能慢一点。

### Q: 时间不对？
**A:** 重新选择一次视频文件试试

### Q: 能不能输入时间直接跳转？
**A:** 目前暂不支持，只能拖动。后续版本可能会加。

## 📏 精度说明

- **时间精度**: 秒级（不精确到毫秒）
- **滑块精度**: 1%（100个刻度）
- **适用视频**: 任意长度（1秒到几小时都可以）

## 🎯 推荐工作流程

```
1. 选择视频 ✓
   ↓
2. 拖动滑块到有代表性的画面
   ↓  
3. 调整字幕的基本参数（大小、位置）
   ↓
4. 拖动到其他场景验证效果
   ↓
5. 微调颜色、轮廓等细节
   ↓
6. 拖动到2-3个不同场景最终确认
   ↓
7. 满意后点"开始执行" ✓
```

## 💯 最佳实践

### 1. 选择3个测试点
建议至少在3个不同位置测试字幕：
- **开头** (0-10%): 看片头效果
- **中间** (40-60%): 看主要内容
- **结尾** (90-100%): 看片尾效果

### 2. 考虑背景对比
- 暗色背景 → 测试亮色字幕
- 亮色背景 → 测试深色轮廓
- 复杂背景 → 测试背景色块

### 3. 避免特殊帧
不要选择这些帧作为预览：
- ❌ 全黑画面（转场）
- ❌ 全白画面（闪光）
- ❌ 画面模糊（运动中）
- ✅ 选择静止、清晰的画面
