# 实时字幕预览快速指南

## 🎯 一句话说明

**导入字幕文件后，拖动时间轴会显示真实的字幕内容！**

## 🚀 快速上手

### 3步操作

```
1️⃣ 选择视频文件
   ↓
2️⃣ 选择字幕文件(.srt)
   ↓ 自动加载
   ↓
3️⃣ 拖动时间轴
   ↓ 实时显示真实字幕！
```

## 📺 界面效果

### 未选择字幕时
```
预览显示: "这是字幕预览效果" (示例文本)
```

### 选择字幕后
```
00:00 → 预览显示: "大家好，欢迎观看..."
00:30 → 预览显示: "今天我们要讲..."
01:00 → 预览显示: "感谢观看！"
```

## 💡 实用场景

### 场景1: 逐句检查
```
设置步进 = 25（约1秒）
连续点击 [+25帧▶]
每句字幕都能看到
发现问题立即调整
```

### 场景2: 快速定位
```
设置步进 = 50
快速浏览找目标台词
如: "Thank you"
改步进 = 1 精确定位
调整该句的样式
```

### 场景3: 多场景验证
```
拖到暗色场景 → 看白色字幕清晰度
拖到亮色场景 → 看黑色轮廓明显度
拖到复杂背景 → 看字幕可读性
```

## ⚙️ 工作原理

```
导入字幕 → 自动解析 → 保存到内存
  ↓
拖动时间轴 → 查找对应字幕 → 渲染预览
  ↓
看到真实字幕内容！
```

## ✅ 特殊情况

| 情况 | 显示结果 |
|------|---------|
| 未选择字幕 | 示例文本 |
| 当前时间有字幕 | **真实字幕** ⭐ |
| 当前时间无字幕 | 纯画面（无字幕）|
| 字幕加载失败 | 示例文本 |

## ⚠️ 注意事项

1. **文件格式**: 必须是 .srt 格式
2. **编码**: 建议 UTF-8 编码
3. **更新延迟**: 拖动后0.5秒更新
4. **无字幕**: 当前时间无字幕时显示纯画面

## 🎉 核心优势

- ⭐ **所见即所得** - 看到的就是最终效果
- ⭐ **真实预览** - 不用猜测，直接看
- ⭐ **高效调整** - 发现问题立即修改
- ⭐ **逐句检查** - 每句字幕都能验证
